# 族谱小程序网站

基于族谱小程序交互图开发的完整网站，数据存储在前端 `data.json` 文件中。

## 功能

- 🌳 **族谱树** - 可视化展示家族谱系结构，点击查看人物详情
- 👥 **族人录** - 家族成员名录列表，支持点击查看详细信息
- 📅 **家族大记事** - 家族事件时间线，支持分类筛选
- 🤖 **AI助手** - 模拟对话式录入界面

## 数据说明

所有数据存储在 `data.json` 文件中，包括：

- `familyTree.persons` - 家族成员信息
- `familyEvents` - 家族大事记
- `aiMessages` - AI助手对话记录

修改 `data.json` 即可更新网站内容。

## 技术栈

- HTML5 + CSS3 + JavaScript
- 无框架依赖，纯原生实现
- 移动端优先设计
